Write this LAST after experiments. Template:

Retrieval-Augmented Generation (RAG) has emerged as a dominant paradigm for grounding LLM outputs in domain-specific documents. However, the choice of embedding model significantly impacts end-to-end system accuracy. In this paper, we present LM Notebook, an AI-powered document notebook employing a RAG pipeline for interactive QA, summarization, quiz generation, flashcard creation, and mind map extraction. We conduct a systematic evaluation of six transformer-based embedding models — spanning BERT, DistilBERT, RoBERTa, and BGE architectures — plus three ensemble pairs using Reciprocal Rank Fusion. 3.1 Research Questions

  • RQ1: How does embedding model choice affect retrieval quality in a RAG pipeline?
  • RQ2: What is the relationship between model size and end-to-end answer accuracy?
  • RQ3: Can ensemble retrieval (two models + RRF) outperform individual models?
  • RQ4: How does chunking strategy interact with embedding model performance?
